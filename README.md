# alarme-carro-system-test
Project of digital logic: Sensor of a security belt developed at system verilog
-This is my first project to implement the first week of "digital design with verilog" from NPTEL/IIT.
-I developed a digital circuit using the "circuit verse" to test the boolean algebra, the gates, input and output.
-After completing, i start my practice at Verilog, so i translate these circuit  to HDL.
-The project is basically an alarm that is trigger of only when the chave(key) it's on car ignition and the cinto(belt) it's not put it on.
-I used the "AND" operator to make 1 . 1 = 1, but since the variable "cinto(belt)" can't be "1", i had to put the "not" operator to change the interpretation of the digital system to change the value of 0 to 1, making 1 . 0' = 1..
-I also used a wave form to see how these variables behave themself in a certain unit of time.
